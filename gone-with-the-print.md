# 紙を共に刷りぬ
## 登場人物
浦川    ジャンク屋の後輩。
三島    ジャンク屋のバイト。生活が困窮している。
森脇    ジャンク屋の先輩。
河村    ジャンク屋によく出入りするチンピラ。
岩崎    印刷局の局員。印刷犯罪を摘発する任務がある。
大橋    名前:大鷲豊継。革命組織「GWP」の代表。自らの顔と身分をAIに奪われ「総議長」として利用されている。

## 全体の流れ

:::
- ジャンク屋3人がプリンターを見つける
- チンピラが金券の偽造を依頼する
- 印刷局員にバレ、印刷テストに挑む
- FAXで暗号が送られてきて、みんなで暗号を解く
- 大橋が出てきて世界がAIに支配されているということがわかる
:::
